<div class="reference-class">


<p> Extends <b><a href="/reference/event"><tt>Event</tt></a></b></p>

<p>The MouseEvent object is received by the <a href="/reference/item"><tt>Item</tt></a>'s mouse event
handlers <a href="/reference/item#onmousedown"><tt>item.onMouseDown</tt></a>, <tt>Item#onMouseDrag</tt>,
<a href="/reference/item#onmousemove"><tt>item.onMouseMove</tt></a>, <a href="/reference/item#onmouseup"><tt>item.onMouseUp</tt></a>, <a href="/reference/item#onclick"><tt>item.onClick</tt></a>,
<a href="/reference/item#ondoubleclick"><tt>item.onDoubleClick</tt></a>, <a href="/reference/item#onmouseenter"><tt>item.onMouseEnter</tt></a> and
<a href="/reference/item#onmouseleave"><tt>item.onMouseLeave</tt></a>. The MouseEvent object is the only parameter passed
to these functions and contains information about the mouse event.</p>

</div>
