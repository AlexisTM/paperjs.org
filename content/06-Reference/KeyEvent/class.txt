<div class="reference-class">


<p> Extends <b><a href="/reference/event"><tt>Event</tt></a></b></p>

<p>The KeyEvent object is received by the <a href="/reference/tool"><tt>Tool</tt></a>'s keyboard
handlers <a href="/reference/tool#onkeydown"><tt>tool.onKeyDown</tt></a>, <a href="/reference/tool#onkeyup"><tt>tool.onKeyUp</tt></a>. The KeyEvent object is
the only parameter passed to these functions and contains information about
the keyboard event.</p>

</div>
